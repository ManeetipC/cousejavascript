ติดตั้ง nodejs และติดตั้งตัวรีเฟสบราวเซอร์ซิ้ง
npm install -g browser-sync

browser-sync เช็คว่ามีตัวบราวเซอร์ซิ้งอยู่ในเครื่องไหม

การเปิด cmd ง่ายๆ โดยเข้าไปที่โฟลเดอร์นั้นๆ แล้วเปลี่ยน url เป็น cmd  แล้วพิม code .

กด ctrl+ตัวเปลี่ยนภาษา จะเปิดหน้าต่าง cmd ใน vs code

ติดตั้ง Beautify เพื่อให้มันจัดโค้ด


File > Perferences > settings แล้วพิมช่องเซิร์ชว่า formatonsave แก้เป็น true เพื่อให้จัดโค้ดอัตโนมัติ

ใน  vs code พิม ! เพื่อสร้างโค้ด html เบื้องต้นให้
พิม dir ใน terminal เพื่อเช็คว่า มีไฟล์ที่เราแก้ไขอยู่รึป่าว
พิม browser-sync start --server . --files . เพื่อรันหน้าเว็บรีโหลด ออโต้
